# Restaurant Ordering System

A web-based Restaurant Ordering System developed using PHP, MySQL, HTML, CSS, Bootstrap, and XAMPP.

##  Features

- Admin Login
- Admin Dashboard
- Category Management (Add, Edit, Delete)
- Food Management (Add, Edit, Delete)
- Food Image Upload
- Dynamic Menu
- Shopping Cart
- Checkout System
- Order Management
- Update Order Status (Pending, Preparing, Completed)

##  Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript
- XAMPP

##  Database

Database Name:

```
restaurant_db
```

##  How to Run the Project

1. Install XAMPP.
2. Copy the project to the `htdocs` folder.
3. Import the database from `database/restaurant.sql`.
4. Start Apache and MySQL.
5. Open:

```
http://localhost/RestaurantOrderingSystem/
```
